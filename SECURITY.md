# 安全策略 Security Policy

## 支持的版本

本项目目前处于初始公开发布阶段，仅对最新的 `main` 分支提供安全修复。

## 报告漏洞

**请勿通过公开 issue、PR 或讨论区披露安全漏洞。**

请通过以下任一私密渠道报告：

- 使用 GitHub 的 **Private vulnerability reporting**（仓库 → Security → Report a vulnerability）；
- 或发送邮件至 `admin@x-ku.org`。

报告时请尽量包含：

- 受影响的组件或接口；
- 复现步骤或 PoC；
- 你评估的影响范围与严重程度。

我们会在确认后尽快跟进，并在修复发布后与你协调披露时间。感谢你以负责任的方式帮助保护本项目的用户。
